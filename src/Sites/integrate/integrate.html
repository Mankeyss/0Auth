Integrate Verify Onto Your Website
In 3 Simple Steps

1

Create a

/0Auth

page and add the following code

<script src="http://localhost:8940/api/v1/datadecoder" data-redirurl="/dashboard" id="dataDecoder"></script></script>

change the

data-redirurl

parameter to a prefered page when the authentication is a success

2

Create a button

<a id="verifyauthbutton" href="http://localhost:8940/authorize?redir=http://localhost:8940/0Auth">Sign In With 0Auth</a>

and don't forget to change the redir parameter to your

/0Auth

page

3

Add the styling for the button in the same file as the button

<link rel="stylesheet" href="http://localhost:8940/api/v1/buttoncss">
